## 总体

| 设计（02 文档） | 当前实现（MVP） |
|----------------|-----------------|
| PostgreSQL + pgvector | SQLite：`data/rag.db` |

## Router JSON：设计 vs 实现

设计稿（01 §5.1）使用 `paths` 数组 + `type` 字段；实现采用扁平结构，便于解析：

```json
{
  "reasoning": "...",
  "paths": {
    "catalog": {"enabled": true},
    "graph": {"enabled": true, "seed_entities": ["关羽"], "hops": 1},
    "hybrid": {"enabled": true, "top_k": 40, "alpha": 0.55, "entity_filter": []}
  },
  "rerank_top_n": 6
}
```

实现见 `rag/router.py` 中 `ROUTER_SYSTEM` 与 `default_plan()`。

## 启动方式（Mac 系统环境）

| 步骤 | 命令 |
|------|------|
| 灌库 | `./ingest.sh` |
| 启动 Web | `./start.sh` |

- 使用系统 `python3`，自动 `source ~/.zshrc` 读取 `DASHSCOPE_API_KEY`
- 不依赖 `.venv`；首次会自动 `pip install --user`
